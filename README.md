# Contextify — AI That Decodes Your Life

Paste any text, screenshot, or document. Get instant plain-English analysis with manipulation detection.

## Features
- Decodes contracts, legal clauses, medical bills
- Detects manipulation: gaslighting, DARVO, pressure tactics
- Flags hidden costs and legal risks
- Analyzes tone and suggests responses
- History with search, filters, favorites

## Run
```bash
flutter pub get
flutter run --dart-define=GROQ_API_KEY=your_key
flutter build web --dart-define=GROQ_API_KEY=your_key
```

## Tech
Flutter + Groq AI + Riverpod + Hive + Material 3
24 Dart files, zero analysis issues.
